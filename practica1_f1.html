Cinemática en una Dimensión

El objetivo de este laboratorio es el de usar conocimientos acerca del movimiento rectilineo uniforme para poder predecir el ganador en una "carrera" de bolas de billar.

Movimiento Rectilineo Uniforme

Cuando se habla de Movimiento Rectilineo Uniforme se asume que se está describiendo el movimiviento de un cuerpo a velocidad constante. Esto quiere decir que, si la velocidad \(v\) se define como $$v = \frac{dx}{dt},$$ donde \(x\) es la distancia recorrida por un objeto y \(t\) el tiempo, entonces: $$x(t) = \int v dt = v \int dt = vt + x_i,$$ con \(x_i\) una constante que representa la posición inicial de un objeto a un tiempo de \(t=0\). Esto quiere decir que si en un experimento medimos la posición \(x(t)\) de un objeto junto con su respectivo tiempo asociado\(t\), la gráfica de los puntos de posición contra tiempo se ajustarían a una línea recta de pendiente \(v\) y con corte en el eje vertical en \(x_i\).

Consideren ahora un ejemplo en el que tres bolas, con tres distintas velocidades \(v_1,v_2\) y \(v_3\) compiten para llegar a una meta. Si consideramos que las velocidades son constantes entonces cada una de las bolas seguirá la ecuación de un Movimiento Rectilíneo uniforme. Ahora bien, dado que en una competencia se quiere saber qué competidor llega primero, a partir de las ecuaciones de movimiento es posible saberlo antes de que cada bola llegue a la meta. Si nosotros medimos las distancias de tal forma que las bolas comenzaron a moverse desde un punto \(x_i = 5\,cm\) a partir del punto desde donde comenzamos a medir y si, por ejemplo, \(v_1 = 2\,m/s,\,v_2= 3\,m/s\) y \(v_3 = 4\,m/s\), entonces las ecuaciones de movimiento de cada bola serán: $$x_1(t) = (2\,m/s)t + 5\,m,\,\,x_2(t) = (3\,m/s)t + 5\,m,\,\,x_3(t) = (4\,m/s)t + 5\,m,$$ con \(x_1,x_2\) y \(x_3\) las posiciones de cada bola respectivamente. Si sabemos que la meta se encuentra a \(10\,m\) desde el punto a partir del cual comenzamos a medir entonces podemos encontrar el tiempo que gastará cada bola en llegar a la meta. Llamando \(t_1,t_2\) y \(t_3\) a los tiempos en los cuales llega cada una de las bolas a la meta entonces, a partir de despejar el tiempo en las ecuaciones de movimiento, con \(x_1=x_2=x_3=10\,m\) entonces: $$t_1 = \frac{10\,m - 5\,m}{2\,m/s} = 2.50\,s,\,\, t_2 = \frac{10\,m - 5\,m}{3\,m/s} = 1.67\,s,\,\,t_3 = \frac{10\,m - 5\,m}{4\,m/s} = 1.25\,s.$$ Como \(t_3 < t_2 < t_1\) entonces la bola que llegará primero a la meta será la bola 3, con una velocidad de \(v_3 = 4\,m/s\). Para la práctica de laboratorio tendrán que hacer algo muy similar, de forma que con este ejemplo se pueden guiar bastante bien respecto a lo que tienen que hacer. ¡Ánimo!.
